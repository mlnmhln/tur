# INSIGHT Travel Agency — лендинг авторских туров Татьяны Звягиной

## Структура

```
tur/
├── index.html   — весь сайт (фото Татьяны встроено в файл)
├── images/      — папка для дополнительных изображений
└── README.md
```

## Открыть
Двойной клик по `index.html`.

## Публикация
GitHub Desktop: Commit to main → Push origin.

## Фоновые фото
Фоновые изображения туров подгружаются по ссылкам с Unsplash (нужен интернет).
Фото Татьяны встроено прямо в index.html и работает офлайн.
